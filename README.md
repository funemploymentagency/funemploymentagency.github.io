# Funemployment Agency — GitHub Pages

This repo hosts the Funemployment Agency landing page.

## Publish
1. Repo name must be `funemploymentagency.github.io`
2. GitHub: Settings → Pages
   - Source: Deploy from a branch
   - Branch: `main` / root
3. Settings → Pages → Custom domain:
   - Set to your domain (e.g. `funemploymentagency.com`)
4. Enable "Enforce HTTPS" once it appears

## Configure application link
Edit `script.js` and replace `GOOGLE_FORM_URL` with your live Google Form URL.
